# Documentation Index

Welcome — this folder contains the developer and operator documentation for the Robotic Face Recognition and Emotion Analysis System.

Files:
- `PROJECT_OVERVIEW.md` — project summary, architecture, features, and high-level notes.
- `MODULE_DOCUMENTATION.md` — per-module and per-function documentation (recommended reading for developers).
- `API_REFERENCE.md` — full HTTP API reference with examples.
- `DEPLOYMENT_GUIDE.md` — installation and run instructions plus troubleshooting tips.

How to use:
- For a quick read, open `PROJECT_OVERVIEW.md`.
- For modifying code, open `MODULE_DOCUMENTATION.md` and find the module you plan to change.
- For running the app, follow `DEPLOYMENT_GUIDE.md`.
